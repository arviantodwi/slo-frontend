# slo-frontend

Front end for the Sports Learning Online

## Prerequisites

- NodeJS 14
- Yarn

## Included Packages

- PostCSS + Autoprefixer
- Babel
- Bootstrap 5
- SASS / SCSS

## How To Use

Install all prerequisites

Clone this project

```bash
yarn install
```

### Use For Development

```bash
yarn serve
```

### Build For Production

```bash
yarn build
```
